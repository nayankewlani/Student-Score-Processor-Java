Student-Score-Processor-Java
java program to process student marks (input from txt file) and generate output in new csv file

A simple Java program that reads student names and scores from a text file,
calculates average scores, and writes the results to a CSV file.

Concepts Used
- Java OOP (classes, constructors, toString)
- File handling (Scanner, PrintWriter)
- Collections (ArrayList)

Input Format
Name,Score1,Score2

Output Format
Name,AverageScore

Future Improvements
- Add input validation
- Use Student class instead of parallel lists
